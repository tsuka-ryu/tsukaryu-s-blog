## はじめに

この記事は日本語フォントやマークダウンの見え方をテストするための記事です。様々な要素を含めて、実際の表示を確認してみましょう。

## 見出しのテスト

### H3見出し

通常の段落テキストです。日本語の文章が自然に表示されているか確認します。**太字のテキスト**や*斜体のテキスト*、そして`インラインコード`などがどのように見えるかをチェックします。

### リストのテスト

順序なしリスト:
- 項目1: これは最初の項目です
- 項目2: 2番目の項目
  - ネストされた項目A
  - ネストされた項目B
- 項目3: 最後の項目

順序ありリスト:
1. 最初のステップ
2. 次のステップ
3. 最後のステップ

## コードブロックのテスト

JavaScriptのサンプルコード:

```javascript
// 日本語コメント付きのコード
function greet(name) {
  console.log(`こんにちは、${name}さん！`);
  return `ようこそ`;
}

const message = greet("太郎");
```

TypeScriptのサンプル:

```typescript
interface User {
  name: string;
  age: number;
  // ユーザーの役割
  role: 'admin' | 'user';
}

const user: User = {
  name: '山田太郎',
  age: 25,
  role: 'user'
};
```

## 引用のテスト

> これは引用文です。日本語の引用がどのように表示されるか確認します。
>
> 複数行の引用も問題なく表示されるはずです。
